Mobile-First Design क्यों?

⚡

Fast Loading

कम से कम data use करते हुए 2G/3G नेटवर्क पर भी तेज़ी से खुले — हर गांव में काम करे।

🔤

हिंदी Typography

बड़े और स्पष्ट Devanagari fonts जो कम पढ़े-लिखे users को भी आसानी से समझ आएं।

👆

Icon-Based Navigation

Text से पहले icons — user को menu पढ़ने की ज़रूरत नहीं, सिर्फ icon देखकर समझे।

🌿

Earthy Color Palette

हरा, केसरिया और मिट्टी के रंग — जो ग्रामीण जीवन से जुड़े हों और भरोसेमंद लगें।

🎨 Design ऐसा क्यों बनाया?

Color Palette: गहरा हरा (Forest Green) और केसरिया (Saffron) — यह दोनों रंग भारत के ग्रामीण जीवन, खेती और ज़मीन से जुड़े हैं। Cream background आंखों को आराम देती है और villagers को familiar लगती है।

Typography: "Tiro Devanagari Hindi" font — यह especially Hindi/Devanagari के लिए designed है और mobile पर clearly readable है। Poppins को supporting font की तरह use किया ताकि English content भी professional लगे।

Mobile-First Approach: Rural users अक्सर low-end smartphones use करते हैं। इसलिए design simple, icon-heavy, और fast-loading है। Grid system automatically mobile पर adjust होता है।

Emotional Connect: Real Hindi copy जो ग्रामीण जीवन की बात करे — "गांव बढ़े, देश बढ़े", hero character (विकास दूत), local news board — सब मिलकर एक familiar और trustworthy feel देते हैं।

Structure: Hero → Services → Leader/Ambassador → Social Posts → Panchayat Board → Mobile View — यह flow user को complete journey देता है जैसे वो पोर्टल explore कर रहा हो।
